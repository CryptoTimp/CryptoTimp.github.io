About

Alpha has been developed by an ex institutional Crypto Trader. Working in a professional firm for over 2 years exposed to Institution trading software solutions an idea was formed to create Alpha. Pulling in aspects from institutional software and adding personal addons that it always lacked, a diverse selection of features and solutions has been created to provide a compact all in one solution for traders. Some concepts used in Alpha have not been previously avaialble to retail traders. These would involve the ladder concept of execution which is paramount for institutiuonal traders to find edge. Usually relying on 1-2s seconds of execution that is only apparent via viewing a ladder for flow rather than a chart. For further explanation please read the features section below.

Features

Ladder / Chart

The Ladder is a depiction of the current state of the orderbook. This runs up and down $100 from the bid/ask to show the user the full depth of market. Here you can see the quantity at each price, supplemented by the threshold setting to highlight prices with greater size. Alongside this to the left of the orderbook is the Size Indicator. This is designed to show levels of importance based on the size present via a bar. To the right of the orderbook is the VAP (Volume Traded at Price) Indicator. This on startup will begin to tally the amount of BTC traded at each price respectively giving the user an idea of value at each price.

There are two types of execution possible via the Ladder. Firstly is directly going to market by using the buy or sell buttons. In addition to this you can click above the bids in the empty blue boxes to go long at market, or coincidingly go short by hitting the empty red boxes below the bid/ask. If you want to post a limit order into the orderbook this can be done also by clicking on a price on the bid or ask and this will highlight your limit in the order column. These limit orders can also be cancelled by double clicking them.

Dashboard

The Dashboard is defined to cover all necessary factors for a trader in one location. The Dashboard is designed so that any panel can be moved anywhere and resized to the users content. This includes a "Pearsons Correlation Matrix" for a large list of major Cryptocurrencies. Alongside this there are "TOS" (Time of Sale) feeds for both Binance and BitMEX Futures.

A full "Order Management Suite" to view, track and close positions can be found on the left hand side. There is a unique widget in the form of the "Risk Ladder" which will display you current % profit/loss in ladder format. A key feature of the Dashboard which is the "Market Watch" panel, here you can view every traded crypto future and its latest statistics. If you want some further comparison the "Biggest Movers" panel can be of use.

Traditional Trading Ladder

A traditional view on trading is now avaialble to Traders via the use of laddering orderbooks. Giving the ability to see further up the bid/ask spread than before.

Instant Execution

Seemless and instant trade execution directly through the exchanges API enables you to get a leg up on other market participants manually entering via the website.

Dashboard

Live and extensive trading dashboard to supplement you execution via the Ladder. The dashboard includes: a Pearsons Correlation matrix, Time of Sales Feeds, Open Positions Table, Open Orders Table, Trade History Table, Risk Ladder, Market Watch and a Biggest Movers Leaderboard.

Binance Compatible

Alpha runs exclusively via Binance USDT Futures for Bitcoin. This is soley because it carries the highest volume of derivative traded BTC currently. Offering the most edge and scenarios to be exploited via Alphas Ladder.

An all in One Suite

Alpha offers an easily accessible platform compiling all the information needed to trade effectively and efficiently into one place.
